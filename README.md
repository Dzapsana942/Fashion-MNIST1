# Fashion-MNIST — klasyfikacja obrazów odzieży

Projekt uczenia maszynowego zrealizowany w Google Colab.

## Cel projektu
Klasyfikacja obrazów odzieży z datasetu Fashion-MNIST przy użyciu dwóch modeli:
- **Dense** (sieć w pełni połączona)
- **CNN** (splotowa sieć neuronowa)

## Co zostało zrobione
- Przygotowanie i normalizacja danych
- Budowa i trening modelu Dense
- Budowa i trening modelu CNN
- Porównanie wyników obu modeli
- Ocena dokładności i analiza błędów

## Technologie
Python, TensorFlow/Keras, Google Colab, NumPy, Matplotlib

## Wyniki
CNN osiągnęło wyższą dokładność niż model Dense 
na zbiorze testowym Fashion-MNIST.
